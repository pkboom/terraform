# Command

```sh
terraform fmt

terraform plan

terraform plan -out output.hcl

terraform plan -var-file variables.tfvars

terraform apply

terraform apply output.hcl // you don't get a prompt for `yes`

terraform apply -var-file variables.tfvars // with a variable file
```
